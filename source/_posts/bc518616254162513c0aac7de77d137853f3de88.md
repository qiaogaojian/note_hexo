## 1. 安装 [^1]

**使用 n 安装和管理 Node** 

安装命令:

```sh
# 安装 npm
sudo apt-get install npm
npm config set registry https://registry.npmmirror.com/

# 安装 yarn
sudo npm i -g yarn
yarn config set registry https://registry.npmmirror.com/

# 安装 n, n 模块是管理 nodejs 版本的
sudo npm i -g n

```
