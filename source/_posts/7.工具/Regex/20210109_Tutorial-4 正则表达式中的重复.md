# 正则表达式中的重复

## 1.正则表达式的分支条件

  正则表达式语法中，表示分支条件的方式为：|

  使用 | 可以将不同的规则分隔开，从左到右的测试每个条件，如果满足了其中的一个分支的  话，就不会再去测试其他的条件了。

  ```sh
  (.[\u4E00-\u9FA5]+)|([\u4E00-\u9FA5]+.)  # 匹配汉字字符
  ```

## 2.正则表达式的重复模式

| 语法 |     说明     |
|:---:|:------------:|
|*|重复0次或更多次|
|+|重复1次或更多次|
|?|重复0次或1次|
|{n}|重复n次|
|{n,}|重复n次或更多次|
|{n,m}|重复n到m次|

## 3.正则表达式中重复的使用场景

这里我们着重列举一个场景，即不同城市间的电话号码的简单的正则匹配：

024-12345678
(012)12345678

```sh
\(\d{3}\)\d{7,8}  #匹配 ()内三位数 + 7/8位数 组成的电话号码
```
