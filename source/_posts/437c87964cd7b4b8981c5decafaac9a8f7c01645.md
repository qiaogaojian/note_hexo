## 1. 安装

**使用 n 安装和管理 Node**

安装命令:

```sh
# 安装 npm
sudo apt-get install npm

# 安装 yarn
sudo npm i -g yarn

# 安装 n, n 模块是管理 nodejs 版本的
sudo npm i -g n

```
