# Nano编辑器的基本用法

## 新建文件

```sh
nano [filename]
```

## 保存文件

```sh
ctrl + o
```

## 复制

```sh
# 单行复制
alt + 6
# 多行复制
ctrl + 6 开始复制
alt + 6  结束复制
```

## 剪切

```sh
# 单行剪切
ctrl + k
# 多行剪切
alt + a  开始剪切
ctrl + k 结束剪切
```

## 粘贴

```sh
ctrl + u
```

## 查找

```sh
ctrl + w
```

## 查找并替换

```sh
ctrl + \ 输入要查找的文本，然后按Enter键，再输入替换后的文本，按Enter
```

## 备份文件

```sh
nano -B [filename]
```
