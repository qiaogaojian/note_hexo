# 常用ADB命令

## 开发

### 查询

- 查看apk对应包名
  ```
  aapt dump badging [app.apk]
  ```
- 查看手机所有应用的包名
  ```
  adb shell pm list packages
  ```
- 查看apk文件的安装位置
  ```
  adb shell pm list packages -f
  ```
