# Git常用命令速查表

## 常用命令

### 1.下载项目

```sh
git clone https://gitee.com/qiaogaojian/MyGames.git
```

### 2.添加更改

```sh
git add .
```

### 3.提交文件

```sh
git commit -m "commit message"
```

### 4.上传代码

```sh
git push
```

### 5.下载代码

```sh
git pull
```

### 6.显示所有分支

```sh
git branch
```

### 7.切换到分支

```sh
git checkout master
```

### 8.新建本地分支

```sh
git checkout -b TestBranch
```

### 9.推送本地分支到远程

```sh
git push origin TestBranch
```
