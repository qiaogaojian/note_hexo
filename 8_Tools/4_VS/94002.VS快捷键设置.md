# VS快捷键设置

## 快捷键

---
| hotkey  | function           |
| ------- | ------------------ |
| alt + d | go to defination   |
| alt + r | find all reference |
| alt + g | find file          |
| alt + t | go to line         |
| alt + f | format file        |
| alt + ; | go to line end     |
| alt + q | nav back           |
| alt + e | nav ahead          |
| alt + c | comment            |
| alt + v | uncomment          |
| alt + a | file struct        |
