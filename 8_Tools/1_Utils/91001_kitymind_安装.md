# kitymind安装

1. 下载项目

    ```sh
    git clone https://github.com/fex-team/kityminder.git
    ```

2. 安装Node
    官网下载安装后,kitymind根目录命令行(git bash)

    ```sh
    npm install
    ```

3. 安装bower

    ```sh
    npm install -g bower
    ```
    安装后:
    ```sh
    bower install
    ```

4. 安装grunt

    ```sh
    npm install -g grunt-cli
    ```

5. 更新

    ```sh
    git submodule init
    git submodule update
    ```

6. 构建

    ```sh
    grunt
    ```
